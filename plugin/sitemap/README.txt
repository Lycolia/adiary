サイト内のすべての記事の permalink を含む、サイトマップを生成します。

Googleに送信する場合、最初に「手動での登録作業が必要」なようですので、
ご注意下さい。
https://www.google.com/webmasters/tools/


生成した xml ファイルの実態は、
	pub/<blogid>/sitemap.xml
に置かれています。

記事を更新するごとに sitemap.xml は再生成されますので、
記事が多いと、処理が重くなるかも知れません。
