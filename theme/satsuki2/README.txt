●テーマ開発ドキュメント
--------------------------------------------------------------------------------
・テーマファイル名に使用可能な文字は「A-Z, a-z, 0-9, _, -」のみです。

・「ID指定」と「クラス指定」のどちらでも設定可能な要素については、
　なるべくクラス指定を使用してください。（必ずではない）

・テーマの最初の行に必ず文字コード宣言を含めてください。
　この書式は厳格である必要があります。
@import "UTF-8";
（参考）http://hp.vector.co.jp/authors/VA022006/css/syntax.html#at-charset-rule

・テーマの最初に「@import url("../base.css");」を記述してください。

・ui-iconは以下のように指定すると自動ロードされます。
　色変更非対応テーマで、直接ロードする場合は pub-dist/ui-icon/ から
　適切な色のアイコンを拾って、テーマディレクトリ内にコピーしてください。
---------------------------------------------
#ui-icon-autoload {
	display:		none;
	background-color:	#0000ff;
}
---------------------------------------------

--------------------------------------------------------------------------------
●スマホ対応
--------------------------------------------------------------------------------

・テーマディレクトリ内に sphone.css というファイルを用意すると、
　スマホ対応テーマになります。

・スマホ表示時のみ sphone.css が「最後に」読み込まれます。

・スマホ非対応テーマでは satsuki2/satsuki2 がスマホ表示用として使用されます。
