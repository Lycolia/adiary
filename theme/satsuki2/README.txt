●テーマ開発、注意事項
--------------------------------------------------------------------------------
・テーマファイル名に使用可能な文字は「A-Z, a-z, 0-9, _, -」のみです。

・テーマの最初の行に必ず文字コード宣言を含めてください。
　この書式は厳格である必要があります。
@import "UTF-8";
（参考）http://hp.vector.co.jp/authors/VA022006/css/syntax.html#at-charset-rule

・テーマの最初に「@import url("../base.css");」を必ず記述してください。

--------------------------------------------------------------------------------
●システムモードの対応について
--------------------------------------------------------------------------------

system_mode_yes というファイルを用意すると、
システムモードでもそのテーマが使われるようになります。


その際は README に
System-mode: yes
と記述してください。
（対応しない場合は no と記述するか、記述しないでください）
